## Getting Started

{Started}

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* npm
  ```sh
  npm install npm@latest -g
  ```

* nvm
  ```sh
  nvm ls
  ```
  ```sh
  nvm use 18.12.0
  ```
  or
  ```sh
  nvm install 18.12.0
  ```

### Installation

1. Clone the repo
   ```sh
   git clone {Repo}
   ```
2. Install NPM packages
   ```sh
   yarn
   ```
3. Build production
   ```sh
   yarn run build
   ```
